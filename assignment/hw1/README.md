# 프로그래밍 과제 1: 반복/재귀로 n! 계산기 (인터랙티브 콘솔)

##  과제 정보
- 과제명: 반복/재귀 n! 계산기
- 작성자: 20190535 김재민
- 작성일: 2025-09-19

## 🎯 학습 목표
∙ 팩토리얼의 수학적 정의와 반복문/재귀 구현 차이를 이해한다.
∙ 메뉴 기반 콘솔 앱을 설계·구현한다.
∙ 간단한 성능 측정(실행 시간)을 통해 구현을 비교한다.

##  실행 방법
1. 저장된 폴더로 이동:
	cd assignment/assignment1
2. 프로그램 실행:
	factorial_app.py

## 메뉴 설명:
	1 : 반복문으로 n! 계산
	2 : 재귀로 n! 계산
	3 : 두 방식 모두 실행 후 결과와 실행 시간 비교
	4 : 준비된 테스트 데이터 일괄 실행
	(테스트 데이터: [0, 1, 2, 3, 5, 10, 15, 20, 30, 50, 100])
	q : 종료

## 참고:
	⦁ factorial_iter(n) : 반복문으로 n! 계산 (n < 0 → ValueError 발생)
	⦁ factorial_rec(n) : 재귀로 n! 계산 (n < 0 → ValueError 발생)
	⦁ run_with_time(func, n) : 함수 실행 결과와 경과 시간 반환
	⦁ 큰 n에서 재귀 실행은 RecursionError가 발생할 수 있으며, 과제 지침에 따라 예외 처리는 크게 하지 않음.
	⦁ 실행 결과는 스크린샷을 통해 참고하시면 됩니다.
