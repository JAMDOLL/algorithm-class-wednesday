## HW2: 도서 관리 프로그램

> 단순 연결 리스트(Singly Linked List)를 활용하여 구현한 도서 관리 프로그램입니다.  
> 사용자 입력을 통해 도서를 추가, 삭제, 조회하고 전체 목록을 출력할 수 있습니다.

### 주요 기능
1. **도서 추가** – 책 번호, 제목, 저자, 출판 연도를 입력하면 리스트에 등록  
2. **도서 삭제** – 책 제목으로 해당 도서를 삭제  
3. **도서 조회** – 책 제목으로 검색하여 도서 정보 출력  
4. **전체 도서 목록 출력** – 등록된 모든 도서 출력  
5. **프로그램 종료**

### 실행 예시
=== 도서 관리 프로그램 ===
1. 도서 추가
2. 도서 삭제 (책 제목으로 삭제)
3. 도서 조회 (책 제목으로 조회)
4. 전체 도서 목록 출력
5. 종료
메뉴를 선택하세요: 1
책 번호를 입력하세요: 1
책 제목을 입력하세요: 해리포터
저자를 입력하세요: J.K.롤링
출판 연도를 입력하세요: 2003
도서 '해리포터'가 추가되었습니다.
